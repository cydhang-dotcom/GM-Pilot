
# GM Pilot 移动端 App 功能清单 (Feature Checklist)

> 版本: v1.6.2
> 更新时间: 2024-01
> 关联文档: [Master PRD](./PRD.md)

## 1. 经营总览 (Dashboard)
- [x] **页面更名**: “经营总览”视觉升级。
- [x] **盈亏看板**: 净利润实时计算与红绿状态区分。
- [x] **AI 智能诊断 (v1.6升级)**:
  - [x] 对话式气泡布局。
  - [x] **无按钮展开**: 采用末尾闪烁省略号点击展开逻辑。
  - [x] 高度锁定，防止打字机渲染时布局跳动。
- [x] **支出结构 (v1.6升级)**: 移除饼图，全量改用 **线性进度条列表**，实现与收入模块的视觉对齐。
- [x] **资金流走势**: 余额默认打码，支持环比走势对比。

## 2. 智能待办 (Inbox)
- [x] **交付时间轴 (v1.6重构)**: 
  - [x] **20px 轴线对齐**: 标题图标与 Marker 垂直中轴线绝对锁定。
  - [x] 拟物化双环 Marker 节点。
- [x] **账单卡片重构 (v1.6.1)**: 
  - [x] **成果导向配色**: 这里的服务费账单已调整为 **翡翠绿 (Emerald)**，与净利润看板对齐。
  - [x] 图标更新为 `ReceiptText`。
- [x] **入职办理中心 (v1.6.2)**:
  - [x] **扫码入职识别**: 列表页与详情页的专属标识与勋章。
  - [x] **信息补全向导**: 针对扫码员工的强制核心信息（薪资/岗位）补录流程。
  - [x] **办理参数化**: 合同/个税/社保办理时的参数配置 Modals。
  - [x] **个税节点**: 新增个税申报独立节点。
- [x] **工资发放确认**: 沉浸式资金大卡片。

## 3. 事务工作台 (Workstation)
- [x] **薪酬/员工/社保/合同**: 完备的 Level 2 列表及数据穿透详情。
- [x] **员工录入 (Employee Add)**: 5步法录入，支持生成自助入职二维码。
- [x] **流水/发票/报销/对账**: 财务全链路管理。
- [x] **凭证管理**: 拟物化记账凭证，支持穿透查看原始影像。

## 4. 企业中心 (Company)
- [x] **隐私模式**: 对公账户金额 **默认打码**。
- [x] **资料管家**: 开票信息一键复制、电子印章状态监控。
