# GM Pilot 移动端 App 功能清单 (Feature Checklist)

> 版本: v1.5.0
> 更新时间: 2023-12
> 关联文档: [Master PRD](./PRD.md)

## 1. 核心看板 (Dashboard) - *Redesigned v1.6*
- [x] **经营周期切换**: 支持月份维度的全盘数据对比 (Dec vs Jan)。
- [x] **盈亏看板 (Hero Card)**: 
  - 净利润实时计算与红绿状态区分。
  - 净利率 (Margin) 监控。
  - 深色高级感主题适配。
- [x] **资金流走势**:
  - 高度与盈亏卡片对齐。
  - 紧凑型布局：大额余额 + 底部走势图。
- [x] **智能风控 (Chat AI)**:
  - 机器人头像 + 气泡消息式布局。
  - 位于成本结构上方。
- [x] **收支结构**:
  - 成本四级分类 (研发/管理/运营/税金) 及进度条占比。
  - 收入来源明细及结算状态。
- [x] **人效监控**:
  - 沉底展示。
  - 在职人数及环比变动。
  - 人均产出 (Revenue Per Capita) 计算。

## 2. 智能待办 (Inbox)
- [x] **交付时间轴 (Timeline)**: *[Moved from Dashboard]*
  - 智能折叠 (Collapsed View)。
  - 节点状态 (Active/Completed/Risk) 及呼吸动效。
  - 嵌入式核心指标 (Metrics) 展示。
- [x] **置顶账单 (Pinned Bill)**: 紧凑卡片展示待支付服务费，一键调起支付。
- [x] **任务聚合 (Grouping)**:
  - **OA 审批**: 聚合内部请假、报销审批流。
  - **入职任务**: 聚合候选人入职确认与归档任务。
- [x] **工资发放确认**:
  - 沉浸式资金卡片 (Wallet Visual)。
  - 资金构成明细 (工资/社保/个税)。
  - 风险提示 (橙色卡片)。
- [x] **审批详情**: OA 单据详情查看及 同意/驳回 操作。

## 3. 事务工作台 (Workstation)

### 3.1 人事管理 (HR)
- [x] **薪酬管理 (`hr-1`)**: 薪资构成饼图、历史发放记录列表、薪资详情 Overlay。
- [x] **员工管理 (`hr-emp`)**: 关键指标统计 (在职/入离)、员工名片、档案详情。
- [x] **五险一金 (`hr-4`)**: 缴纳总额卡片、增减员变动明细、凭证下载。
- [x] **合同管理 (`hr-6`)**: 到期预警统计、合同列表、续签提醒。

### 3.2 财务运营 (Finance)
- [x] **对账中心 (`fn-rec`)**: 同步健康度评分、差异处理清单、异常沟通 Chat。
- [x] **资金流水 (`fn-flow`)**: 银行流水列表、收支详情、业务用途补充。
- [x] **发票管理 (`fn-4`)**: 进项/销项统一列表、扫码/开票入口、发票拟物化详情。
- [x] **费用报销 (`fn-reim`)**: 报销发起入口、证据链缺失预警、报销单详情。
- [x] **凭证管理 (`fn-3`)**: 
  - 月结进度条 (Stepper)。
  - 凭证列表及详情 (会计分录)。
  - 原始单据穿透 (结构化数据 + 影像预览)。
  - 月结验收流程。
- [x] **财税报表 (`fn-5`)**: 经营趋势图穿透、报表文件 (PDF/Excel) 预览下载。
- [x] **税款缴纳 (`fn-2`)**: 申报倒计时、税种明细、申报状态追踪。

### 3.3 专项服务 (Services)
- [x] **政府补助 (`srv-subsidy`)**: 累计获批金额统计、补助申请进度追踪。
- [x] **服务请求 (`ot-*`)**: 专属顾问卡片、服务工单发起与记录查看。

## 4. 企业中心 (Company)
- [x] **企业名片**: 实名认证/存续状态、统一信用代码复制。
- [x] **对公账户**: 拟态银行卡设计、余额隐私模式 (Eye Toggle)。
- [x] **常用资料**:
  - **开票信息**: 一键复制全部。
  - **收件地址**: 地址管理。
  - **权益资产**: 电子印章与数字证书状态。
