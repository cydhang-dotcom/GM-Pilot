# GM Pilot 移动端 App 功能清单 (Feature Checklist)

> 版本: v2.2.0
> 更新时间: 2024-01
> 关联文档: [Master PRD](./PRD.md)

## 1. 经营总览 (Dashboard)
- [x] **盈亏看板**: 净利润实时计算与红绿状态区分。
- [x] **AI 智能诊断 (GenAI)**: 采用 Gemini Streaming API 实现流式诊断。
- [x] **支出结构**: 线性进度条列表，类别色点对齐 20px 轴线。
- [x] **资金流走势**: 支持实时同步银行数据及环比对比。

## 2. 智能待办 (Inbox)
- [x] **交付时间轴**: 像素级对齐，实时反馈外包服务履约状态。
- [x] **入职办理中心**: 支持扫码入职、资料补全及参数化办理。
- [x] **OA 审批**: 报销合规预检，风险点自动高亮。

## 3. 事务工作台 (Workstation)
- [x] **薪酬管理 (v2.2.0)**: 
  - [x] 月度工资总表确认。
  - [x] **个人明细穿透**: 点击进入个人工资条，详细拆解应发、扣除（个税/社保）及实发金额。
- [x] **五险一金 (v2.2.0)**: 
  - [x] **办理状态可视化**: 动态显示办理日期（已完成）或预计办理日期（办理中）。
  - [x] **回单管理**: 支持 PDF 拟物化回单预览。
- [x] **员工管理**: 档案全景视图，支持原位脱敏。
- [x] **凭证管理**: 拟物化记账凭证，支持月结验收。

## 4. 企业中心 (Company)
- [x] **隐私模式**: 开启后对公账户及薪资等敏感数值默认打码。
- [x] **资料管家**: 开票信息、收件地址、电子印章管理。
