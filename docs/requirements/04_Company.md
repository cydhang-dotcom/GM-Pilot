# 模块需求: 我的企业 (Company)

> **优先级**: P1
> **涉及页面**: `pages/Company.tsx`

## 1. 概述
企业数字资产管理中心，强调隐私保护与资产展示。

## 2. 功能组件拆解

### 2.1 企业数字名片 (Identity)
- [x] **信息展示**: Logo、全称、实名状态标签、统一社会信用代码。
- [x] **快捷操作**: 提供一键复制信用代码、二维码查看。

### 2.2 对公账户卡片 (Bank Account)
- [x] **拟态设计**: 模拟物理银行卡视觉，使用深色渐变背景与玻璃质感。
- [x] **隐私默认保护 (Privacy by Default)**:
  - **初始状态**: 用户进入页面时，账户余额必须显示为打码模式 (`¥ 888,888.88` + `blur` 滤镜)。
  - **切换交互**: 点击眼睛图标实现明文/打码切换。
- [x] **收支快照**: 底部显示本月累计入账及出账的简要数字。

### 2.3 常用功能菜单
- [x] **Overlay 交互**: 所有二级入口（开票信息、地址、印章）均使用全屏覆盖层弹出，保持底层状态。
