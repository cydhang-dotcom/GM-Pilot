# 模块需求: 我的企业 (Company)

> **优先级**: P1
> **涉及页面**: `pages/Company.tsx`

## 1. 概述
企业数字资产管理中心，强调隐私保护与资产展示。所有详情查看均采用当前页面覆盖 (Overlay) 模式，保持上下文。

## 2. 功能组件拆解

### 2.1 企业数字名片 (Identity)
展示企业法定身份信息。
- [x] **状态展示**: 实名认证与存续状态标签。

### 2.2 对公账户卡片 (Bank Account)
核心资金视图，支持隐私模式。
- [x] **隐私切换**: 点击“眼睛”图标切换余额显示/隐藏 (`******`)。

### 2.3 常用功能菜单 (Menu Groups)
结构化的功能资源入口，点击对应项进入 Level 3 详情页：
- [x] **开票信息 (`InvoiceInfoDetail`)**: 大字展示税号、地址、电话、开户行，支持一键复制。
- [x] **收件地址 (`AddressDetail`)**: 管理公司前台及各部门收件地址。
- [x] **权益与资产 (`AssetDetail`)**: 展示电子印章列表及数字证书有效期。
