# 功能需求: OA 审批详情 (OA Workflow)

> **所属模块**: Inbox
> **优先级**: P1
> **涉及页面**: `pages/Inbox.tsx`

## 1. 需求描述
处理内部行政审批流（请假、报销、采购），并查看自己发起的申请状态。

## 2. 功能清单 (Checklist)
- [x] **审批列表**: 在聚合视图中展示待审批项，包含申请人、类型、摘要。
- [x] **审批详情 (Overlay)**: 
  - 点击单条任务弹出 `OADetail`。
  - 展示申请人头像、申请金额、事由。
- [x] **决策操作**: 
  - 底部固定双按钮：“驳回” (次要) 与 “同意” (主要)。
