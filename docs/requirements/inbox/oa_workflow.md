# 功能需求: OA 审批详情 (OA Workflow)

> **所属模块**: Inbox
> **优先级**: P1

## 1. 需求描述
处理内部行政审批流（请假、报销、采购），并查看自己发起的申请状态。

## 2. 功能清单 (Checklist)
- [ ] **角色切换**: 顶部 Tab 切换 “待我审批” (Approver) 与 “我发起的” (Requester)。
- [ ] **状态筛选**:
  - 审批视图: 待处理 / 已处理。
  - 发起视图: 审批中 / 已通过 / 未通过 / 已撤销。
- [ ] **审批卡片**:
  - 显示申请人、类型（图标区分）、摘要。
  - 显示“耗时”提示（如：等待您处理已耗时 2小时）。
- [ ] **流程节点**: 在“我发起的”视图中，显示当前处于哪个节点（如：财务总监审批中）。
