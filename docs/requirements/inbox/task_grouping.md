# 功能需求: 任务聚合与列表 (Task Grouping)

> **所属模块**: Inbox
> **优先级**: P1

## 1. 需求描述
避免大量同质化小任务（如10个人的入职确认）刷屏，影响重要任务的查找。

## 2. 功能清单 (Checklist)
- [ ] **入职聚合**: 来源为 HR Outsourcing 的入职/增员任务聚合为一个文件夹。
- [ ] **OA聚合**: 来源为 Internal OA 的审批任务聚合为一个文件夹。
- [ ] **文件夹信息**: 显示任务数量、预览文本（如“包含 张三、李四 等”）。
- [ ] **普通列表**: 无法聚合的高优任务（如工资确认、税金申报）按优先级排序平铺展示。
- [ ] **优先级视觉**: High Priority 任务需有红点或高亮标识。
