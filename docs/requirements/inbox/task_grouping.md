# 功能需求: 任务聚合与列表 (Task Grouping)

> **所属模块**: Inbox
> **优先级**: P1
> **涉及页面**: `pages/Inbox.tsx`

## 1. 需求描述
避免大量同质化小任务（如10个人的入职确认或OA审批）刷屏，影响重要任务的查找。

## 2. 功能清单 (Checklist)
- [x] **聚合卡片**: 
  - **OA 审批**: 聚合报销、请假等内部流程。
  - **入职任务 (New)**: 聚合待入职员工的确认与归档任务。
  - 显示包含的任务数量红点及摘要文本。
- [x] **二级视图 (Group View)**: 
  - 点击聚合卡片，当前页面切换为该组的任务列表（Slide In 动效）。
  - **入职列表**: 显示候选人、部门、职位及当前进度状态。
  - 顶部带返回按钮，退出组视图。
- [x] **普通列表**: 无法聚合的高优任务（如工资确认）按优先级排序平铺展示。
