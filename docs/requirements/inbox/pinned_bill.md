# 功能需求: 账单置顶卡片 (Pinned Bill)

> **所属模块**: Inbox
> **优先级**: P0

## 1. 需求描述
当有涉及资金支付的高优先级任务（如服务费账单）时，需在列表顶部以大卡片形式展示，确保不被遗漏。

## 2. 功能清单 (Checklist)
- [ ] **触发条件**: 任务类型为 `CONFIRM` 且涉及金额支付。
- [ ] **视觉强调**: 使用独立背景色或渐变色，区别于普通列表项。
- [ ] **核心信息**: 显示账单标题、服务周期、截止时间。
- [ ] **金额展示**: 大字号显示待支付金额。
- [ ] **快捷操作**: 提供“立即支付”按钮，点击触发支付流程。
