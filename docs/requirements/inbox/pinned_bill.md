# 功能需求: 账单置顶卡片 (Pinned Bill)

> **所属模块**: Inbox
> **优先级**: P0
> **涉及页面**: `pages/Inbox.tsx`

## 1. 需求描述
当有涉及资金支付的高优先级任务（如服务费账单）时，需在列表顶部以紧凑卡片形式展示，确保不被遗漏且与下方模块视觉统一。

## 2. 功能清单 (Checklist)
- [x] **触发条件**: 任务类型为 `CONFIRM` 且涉及金额支付。
- [x] **视觉强调**: 使用渐变背景 (Indigo-to-White) 与阴影，区别于普通列表项，但保持单行紧凑布局。
- [x] **核心信息**: 显示账单标题、周期、金额。
- [x] **详情交互 (Overlay)**: 
  - 点击卡片弹出 `BillDetail` 覆盖层。
  - 详情页展示详细的费用构成、收款方信息。
  - 底部固定“立即支付”按钮。
