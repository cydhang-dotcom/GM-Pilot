# 模块需求: 核心看板 (Dashboard)

> **优先级**: P0
> **涉及页面**: `pages/Dashboard.tsx`
> **设计风格**: Financial Pilot (盈亏视角, 决策导向)

## 1. 概述
Dashboard 是总经理的“经营驾驶舱”，从单纯的支出记录转变为 **盈亏视角 (Profit & Loss)**。核心目标是让 GM 一眼看懂本月赚了多少、钱花哪了、有无风险。

## 2. 功能组件拆解

### 2.1 顶部导航与周期切换 (Header & Toggle)
- [x] **周期切换**: 提供月份切换器 (如 `2023-12` vs `2024-01`)，支持不同经营周期的快速对比。
- [x] **数据联动**: 切换月份时，页面所有指标（利润、收入、成本、风险）需同步刷新。

### 2.2 核心盈亏看板 (Hero Card)
- [x] **净利润计算**: 实时展示 `总收入 - 总支出` 的结果。
- [x] **视觉反馈**: 
  - **盈利**: 翠绿色 (Emerald) 渐变背景。
  - **亏损**: 玫瑰红 (Rose) 渐变背景。
  - *v1.6 更新*: 改为深色岩石灰 (Dark Slate) 高级感主题，仅数字带色。
- [x] **净利率监控**: 显示 Net Profit Margin (%)。

### 2.3 资金流走势 (Cash Flow) - *Compacted*
- [x] **高度对齐**: 卡片高度与上方“净利润”卡片保持一致，形成整齐的视觉块。
- [x] **紧凑布局**: 
  - 左上角展示“当前余额”大数字。
  - 走势图铺满卡片剩余空间，移除多余留白。
- [x] **趋势对比**: 保留本月与上月余额走势的对比图。

### 2.4 智能风险诊断 (Smart Risk Monitor) - *Chat AI Style*
- [x] **拟人化设计**: 移除传统卡片标题，改为“机器人头像 + 气泡消息”布局。
- [x] **第一人称**: 使用“我为您发现...”的口吻陈述风险。
- [x] **位置调整**: 移动至“成本结构”上方，模拟对话流。

### 2.5 收支结构分析 (Structure Analysis)
- [x] **Tab 切换**: 支持在“成本结构”与“收入构成”之间切换。
- [x] **成本深钻**: 
  - 分类展示：研发 (R&D)、管理 (Admin)、运营 (Ops)、税金 (Tax)。
  - 进度条可视化占比。
- [x] **收入追踪**: 列出具体回款项目、日期及入账状态。

### 2.6 人效运营指标 (Efficiency Card) - *Moved to Bottom*
- [x] **位置调整**: 移至页面最底部，作为次级指标。
- [x] **组合展示**: 在职人数与人均产出合并为一个简约卡片。
