# 功能需求: 支出构成分析 (Expenditure Breakdown)

> **所属模块**: Dashboard
> **优先级**: P1
> **涉及页面**: `pages/Dashboard.tsx`

## 1. 需求描述
让管理者直观感知本月钱花哪儿了，特别是刚性的人力成本与税费。

## 2. 功能清单 (Checklist)
- [x] **饼图可视化 (Pie Chart)**: 
  - **维度**: 工资薪金 (蓝)、五险一金 (浅蓝)、纳税总额 (橙)、外包服务费 (紫)。
  - **交互**: 点击扇区或图例可查看具体金额 (Tooltip)。
- [x] **数据展示**:
  - 图例侧边排列，显示科目名称及金额缩略 (如 ¥32.5w)。
- [x] **跳转入口**: 点击箭头跳转至“财税报表” (`fn-5`) 详情页。
