# 功能需求: 支出构成分析 (Expenditure Breakdown)

> **所属模块**: Dashboard
> **优先级**: P1
> **涉及页面**: `pages/Dashboard.tsx`

## 1. 需求描述
让管理者直观感知本月钱花哪儿了，重点关注研发、管理、运营及税金的比例。

## 2. 功能清单 (Checklist)
- [x] **线性进度条列表 (Linear Progress Bars)**: 
  - **取代饼图**: 移除圆形图表，改用扁平化列表以适应移动端纵向布局。
  - **颜色映射**: 研发 (Indigo)、管理 (Purple)、运营 (Amber)、税金 (Slate)。
- [x] **关键信息**:
  - 显示二级分类名称（如：管理费用）及主要子项摘要（如：办公费、房租）。
  - 右侧显示金额 (Font Mono)。
- [x] **动效**: 页面进入时进度条从 0% 加载至目标值。
- [x] **跳转入口**: 点击右上角箭头跳转至“财税报表” (`fn-5`)。
