# 功能需求: 人员规模趋势 (Personnel Trend)

> **所属模块**: Dashboard
> **优先级**: P2
> **涉及页面**: `pages/Dashboard.tsx`

## 1. 需求描述
展示人力规模变动趋势，监控入离职流动情况。

## 2. 功能清单 (Checklist)
- [x] **核心指标**: 
  - 大字展示当前在职人数。
  - 标签展示本月 **入职 (+)** (绿) 和 **离职 (-)** (红) 人数。
- [x] **趋势图 (Area Chart)**: 
  - 展示近 4 个月的人数变化面积图，绿色渐变填充。
  - 交互: 支持 Tooltip 查看历史月份具体人数。
- [x] **跳转**: 点击箭头跳转至“员工管理” (`hr-emp`) 详情页。
