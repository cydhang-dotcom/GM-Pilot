# 功能需求: 交付进度时间轴 (Timeline)

> **所属模块**: Dashboard
> **优先级**: P0
> **涉及页面**: `pages/Dashboard.tsx`

## 1. 需求描述
按时间顺序展示本月关键业务节点，让用户清晰了解当前进度。

## 2. 功能清单 (Checklist)
- [x] **节点状态展示**: 
  - **Completed (绿)**: 已完成。
  - **Active (蓝)**: 进行中，显示当前处理的具体细节。
  - **Pending (灰)**: 待开始。
  - **Risk (橙)**: 存在异常或风险 (如：待办超时)。
- [x] **智能折叠 (Collapsed View)**:
  - 默认仅展示 **进行中 (Active)** 或 **有风险 (Risk)** 的节点，保持首屏简洁。
  - 点击头部标题或“查看全部”展开完整时间轴。
- [x] **结构化内容**:
  - **关键指标 (Metrics)**: 针对薪酬/人员变动，直接在时间轴卡片内展示核心数据（如实发总额、趋势箭头）。
  - **子项清单 (Sub Items)**: 针对税务/补助，列出具体细项的状态。
  - **状态图标 (Status Icons)**: 子项列表左侧显示完成状态图标（绿勾、蓝表、橙警示），替代原有圆点，增强视觉识别。
- [x] **交互**:
  - 点击“查看详情”跳转至对应的 Level 2 业务模块。
  - 风险项支持直接点击跳转处理。
- [x] **进度概览**: 标题栏右侧显示整体进度百分比 (如 25%)。
