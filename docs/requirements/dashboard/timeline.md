# 功能需求: 交付进度时间轴 (Timeline)

> **所属模块**: Dashboard -> Inbox (现已迁移至 Inbox)
> **优先级**: P0
> **涉及页面**: `pages/Inbox.tsx`

## 1. 需求描述
按时间顺序展示本月关键业务节点，作为外包履约的“指挥棒”。

## 2. 视觉规格 (High Fidelity Specs)
- [x] **像素级对齐**: 
  - **中轴线锁定**: 标题图标（Clock）与下方所有节点 marker 的几何重心必须完全锁定在左侧 **20px** 的垂直轴线上。
  - **容器规格**: 图标与 Marker 均居中定位在 `w-10` (40px) 的透明容器内。
- [x] **拟物化节点**:
  - **Active 状态**: 双环设计，外环为实线边框，内环为实心点且带呼吸灯效果。
  - **Pending 状态**: 灰色虚线边框环，内部留白。
- [x] **连接线**: 根据节点状态变色（Completed 绿 / Active 蓝 / Pending 灰）。

## 3. 功能特性
- [x] **智能折叠**: 默认仅展示 Active 节点。
- [x] **内嵌指标**: 在薪酬节点卡片内展示实发金额与人数变动。
