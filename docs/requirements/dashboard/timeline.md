# 功能需求: 交付进度时间轴 (Timeline)

> **所属模块**: Dashboard
> **优先级**: P0

## 1. 需求描述
按时间顺序展示本月关键业务节点，让用户清晰了解当前进度。

## 2. 功能清单 (Checklist)
- [ ] **节点状态展示**: 
  - Completed (绿): 已完成。
  - Active (蓝): 进行中，显示当前处理的具体细节。
  - Pending (灰): 待开始。
  - Risk (橙): 存在异常或风险。
- [ ] **默认折叠**: 仅展示 Active (进行中) 或 Risk (风险) 节点，节省首屏空间。
- [ ] **展开全部**: 点击头部标题或底部“查看全部”按钮展开完整时间轴。
- [ ] **进度概览**: 标题栏右侧显示整体进度百分比 (如 25%)。
- [ ] **操作内嵌**: 节点内部支持 Action Button (如“去确认”)，点击跳转 Inbox。

## 3. 数据来源
- 外包服务商履约系统状态同步。
