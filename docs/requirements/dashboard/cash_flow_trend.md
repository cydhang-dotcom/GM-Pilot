# 功能需求: 现金流走势环比 (Cash Flow Trend)

> **所属模块**: Dashboard
> **优先级**: P0
> **设计核心**: 基准对比 (Benchmarking)、流动性预测 (Liquidity Forecasting)

## 1. 用户故事 (User Story) 与 数据逻辑

### US1: 烧钱速度对齐
*   **数据业务逻辑**: 
    *   **环比对齐**: 横轴按“日”进行 1:1 对齐。
    *   **风险预警**: 若本月实时余额线连续 3 天低于上月同期线且差额扩大 20% 以上，触发 AI 诊断“流动性收紧”提示。

## 2. 界面行为规范 (UI Behaviors)

*   **图表标准**: 本月实线 (#3B82F6) + 上月灰色虚线 (#D1D5DB)。
*   **交互逻辑**: Tooltip 必须在手指滑动时实时计算并显示“差额金额”与“跌幅百分比”。

## 3. 验收标准 (Acceptance Criteria)

- [x] 开启隐私模式时，图表纵坐标数值及 Tooltip 数值必须高斯模糊。
- [x] 切换月份时，图表应具备平滑的路径过渡动画。
