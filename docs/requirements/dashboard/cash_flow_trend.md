# 功能需求: 现金流走势环比 (Cash Flow Trend MoM)

> **所属模块**: Dashboard
> **优先级**: P1
> **涉及页面**: `pages/Dashboard.tsx`

## 1. 需求描述
总经理需要直观对比本月与上月同期的现金流（余额）走势，以快速判断资金消耗速度是否异常。

## 2. 功能清单 (Checklist)
- [x] **对比图表 (Comparison Area Chart)**: 
  - **本月 (蓝色实线)**: 展示截至目前的余额走势。
  - **上月 (灰色虚线)**: 展示上月同期的走势作为基准线。
  - 隐藏 Y 轴具体数值，仅展示趋势形态，聚焦于“快慢”与“高低”。
- [x] **智能交互 (Tooltip)**:
  - 悬浮显示某日的 **本月/上月对比金额**。
  - 自动计算并显示 **差额** 与 **涨跌幅百分比**。
- [x] **摘要展示**:
  - 标题栏展示整体环比跌幅 (如 -2.3%)。
- [x] **入口**: 点击跳转至“资金流水” (`fn-flow`) 详情页。
