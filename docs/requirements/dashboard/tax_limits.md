# 功能需求: 税前扣除限额监控 (Tax Limits)

> **所属模块**: Dashboard
> **状态**: ⏸️ 已挂起 (Backlog)
> **说明**: 为保持 Dashboard 移动端首屏的简洁性，该功能已在 v1.0 版本中移除，相关数据可在“财税报表”中查看。

## 1. 需求描述 (已归档)
监控有税法扣除限额的科目（如业务招待费、职工教育经费等），防止超支导致汇算清缴时的纳税调增。

## 2. 原定功能清单
- [ ] **科目卡片**: 网格布局展示各科目。
- [ ] **进度条**: 可视化展示 已用金额 / 限额标准。
- [ ] **状态预警**:
  - **Safe (绿)**: 使用率 < 80%。
  - **Warning (橙)**: 使用率 > 80%。
  - **Critical (红)**: 使用率 > 100% (提示“需纳税调整”)。
