# 功能需求: 税前扣除限额监控 (Tax Limits)

> **所属模块**: Dashboard
> **优先级**: P0 (风险控制核心)

## 1. 需求描述
监控有税法扣除限额的科目（如业务招待费、职工教育经费等），防止超支导致汇算清缴时的纳税调增。

## 2. 功能清单 (Checklist)
- [ ] **科目卡片**: 网格布局展示各科目。
- [ ] **进度条**: 可视化展示 已用金额 / 限额标准。
- [ ] **状态预警**:
  - **Safe (绿)**: 使用率 < 80%。
  - **Warning (橙)**: 使用率 > 80%。
  - **Critical (红)**: 使用率 > 100% (提示“需纳税调整”)。
- [ ] **数据计算**: 限额标准需根据当年累计收入动态计算（如招待费为收入的5‰）。
