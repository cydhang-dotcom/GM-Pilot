# 功能需求: 常用功能菜单 (Menu Groups)

> **所属模块**: Company
> **优先级**: P1
> **涉及页面**: `pages/Company.tsx`

## 1. 菜单结构
- [x] **常用业务资料**: 开票信息、收件地址。
- [x] **权益与资产**: 电子印章、数字证书、签约服务。
- [x] **企业管理**: 成员权限、操作日志、消息通知。

## 2. 详情交互 (Level 3 Overlay)
- [x] **开票信息详情**: 大字展示税号、地址、电话、账号，支持一键复制全部。
- [x] **收件地址详情**: 卡片展示地址信息，标记“默认”地址。
- [x] **资产详情**: 列表展示印章名称及状态，数字证书有效期。
