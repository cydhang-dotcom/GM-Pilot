# 功能需求: 对公账户卡片 (Bank Account)

> **所属模块**: Company
> **优先级**: P0
> **涉及页面**: `pages/Company.tsx`

## 1. 需求描述
提供类似银行 App 的资产卡片体验，支持在公共场合保护隐私。

## 2. 功能清单 (Checklist)
- [x] **视觉设计**: 拟态银行卡样式，深蓝色渐变背景。
- [x] **账户信息**: 显示开户行（招商银行）、账户性质。
- [x] **隐私模式 (Privacy)**:
  - 点击“眼睛”图标切换显示。
  - **隐藏状态**: 余额显示 `******`，卡号显示后四位。
  - **显示状态**: 显示真实余额与完整卡号。
- [x] **收支概览**: 底部显示本月累计收入 (绿箭头) 与支出 (红箭头)。
