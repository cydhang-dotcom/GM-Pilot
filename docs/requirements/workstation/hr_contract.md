# 功能需求: 合同管理详情 (Contract)

> **入口 ID**: `hr-6`
> **优先级**: P2
> **涉及页面**: `pages/workstation/hr/Contract.tsx`

## 1. 看板视图 (Level 2)
- [x] **统计概览**: 展示有效合同数及即将到期数。
- [x] **预警列表**:
  - **即将到期 (Expiring)**: 橙色侧边条高亮，显示剩余天数及动效。
  - **正常状态**: 蓝色侧边条，显示到期日期。

## 2. 合同详情 (Level 3 Overlay)
- [x] **倒计时**: 针对即将到期的合同，醒目展示剩余天数及截止日期。
- [x] **续签提醒**: 橙色卡片提示管理建议。
- [x] **操作**: 底部提供“不再续签”与“发起续签”决策按钮。
