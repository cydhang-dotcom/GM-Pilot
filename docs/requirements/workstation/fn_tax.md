# 功能需求: 税款缴纳详情 (Tax)

> **入口 ID**: `fn-2`
> **优先级**: P1
> **涉及页面**: `pages/workstation/finance/Tax.tsx`

## 1. 看板视图 (Level 2)
- [x] **申报倒计时**: 顶部卡片提示距离本期申报截止天数，提供“立即申报”入口。
- [x] **纳税记录**: 列表展示各税种（增值税、个税）的金额、状态（申报中/已缴款）。

## 2. 税款详情 (Level 3 Overlay)
- [x] **金额卡片**: 大字展示本期应缴总额及所属期。
- [x] **税种明细**: 列表展示增值税、城建税、教育费附加的具体金额构成。
- [x] **申报进度**: 
  - 垂直时间轴展示：生成申报表 -> 等待确认/扣款成功。
  - 状态为“申报中”时，底部显示“确认申报并扣款”按钮。
