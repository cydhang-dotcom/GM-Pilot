# 功能需求: 员工管理详情 (Employee)

> **入口 ID**: `hr-emp`
> **优先级**: P1
> **涉及页面**: `pages/workstation/hr/Employee.tsx`

## 1. 看板视图 (Level 2)
- [x] **关键统计**: 顶部三栏展示 在职总数、本月入职 (+)、本月离职 (-)。
- [x] **搜索过滤**: 提供搜索框，支持按姓名/部门快速查找。
- [x] **员工列表**: 卡片展示头像、姓名、部门职位、入职日期及状态标签（正式/试用）。

## 2. 员工档案 (Level 3 Overlay)
- [x] **个人名片**: 顶部大头像、姓名职位，提供“呼叫”与“邮件”快捷操作。
- [x] **分组信息**:
  - **基本信息**: 工号、入职日期、身份证号（脱敏）。
  - **合同信息**: 合同类型、到期日。
- [x] **状态标识**: 清晰标记“试用期”或“正式员工”状态颜色。
