# 功能需求: 资金流水详情 (Cash Flow)

> **入口 ID**: `fn-flow`
> **优先级**: P1
> **涉及页面**: `pages/workstation/finance/CashFlow.tsx`

## 1. 流水列表 (Level 2)
- [x] **筛选器**: 顶部提供横向滚动筛选（全部/待解释/收入/支出）。
- [x] **列表项**:
  - 左侧展示收/支类型图标。
  - 右侧展示金额（收入绿/支出黑）。
  - **状态提示**: “需解释”或“需确认”状态需有左侧橙色色条标记。
  - **底部操作区**: 直接在列表卡片底部提供快捷操作按钮（如“补充用途”、“确认匹配”）。

## 2. 流水详情 (Level 3 Overlay)
- [x] **金额大卡片**: 
  - 顶部展示超大号金额，明确收支方向。
  - 显示交易对手名称及摘要。
- [x] **信息明细**: 展示交易时间、对方账号、交易单号。
- [x] **电子回单**: 提供查看银行电子回单附件的入口。
- [x] **业务处理区**:
  - **待解释**: 提供常用用途标签（销售回款、服务费等）供快速选择。
  - **已匹配**: 展示已关联的单据编号（如发票号）。
