# 功能需求: 电子凭证详情 (Voucher Manager)

> **入口 ID**: `fn-3`
> **优先级**: P2
> **涉及页面**: `pages/workstation/finance/VoucherManager.tsx`

## 1. 关账进度看板 (Level 2 Header)
- [x] **进度条**: 可视化展示当前月结进度（智能做账 -> 人工复核 -> 纳税申报 -> 月结验收 -> 归档锁定）。
- [x] **主行动点**: 根据当前进度状态，动态显示主按钮（如“开始验收锁版”或“数据处理中”）。

## 2. 凭证列表 (Level 2 List)
- [x] **清单展示**: 显示凭证号、摘要、金额、附件数量及关联方。
- [x] **交互**: 点击进入凭证详情 Overlay。

## 3. 凭证详情 (Level 3 Overlay)
模拟真实纸质凭证的视觉体验。
- [x] **凭证卡片**: 
  - 拟物化设计（锯齿边缘/虚线）。
  - **会计分录**: 表格化展示借贷方科目及金额。
- [x] **附件挂载**: 
  - 列表展示该凭证挂载的所有原始单据（发票、回单、收据）。
  - 不同类型附件使用不同图标/颜色区分。

## 4. 单据穿透 (Level 4 Overlay)
点击附件列表中的单项，进入源单据详情。
- [x] **系统结构化数据**: 
  - 针对发票：展示票面关键信息（购买方、销售方、税额）。
  - 针对流水：展示收付方及交易摘要。
- [x] **原始影像入口**: 提供查看原始文件缩略图的入口。

## 5. 影像预览 (Level 5 Overlay)
- [x] **大图预览**: 全黑背景沉浸式查看原始单据（PDF/图片）。

## 6. 月结验收流程 (Acceptance View)
- [x] **清单确认**: 用户需逐项勾选确认（如“资金调节表平衡”、“交付物齐套”）。
- [x] **锁版归档**: 确认后系统锁定当前账期数据，不可篡改。
