# 功能需求: 政府补助详情 (Government Subsidy)

> **入口 ID**: `srv-subsidy`
> **优先级**: P2
> **涉及页面**: `pages/workstation/services/GovernmentSubsidy.tsx`

## 1. 概述
帮助总经理追踪政府补助（如稳岗补贴、高新认定奖励）的申请进度及资金到账情况。

## 2. 核心看板 (Summary)
- [x] **累计获批金额**: 展示本年度累计已获批的补助总额。
- [x] **资金状态分布**: 区分“已到账”与“审核中”的金额，让资金预期更准确。

## 3. 补助清单 (Subsidy List)
- [x] **状态标签**:
  - **已到账 (Success)**: 资金已入账。
  - **审核中 (Info)**: 正在走流程。
  - **待申请 (Gray)**: 符合条件但尚未提交。
- [x] **关键信息**: 补助名称、类型（就业补贴/资质认定）、金额。

## 4. 详情页 (Level 3 Detail)
通过 Overlay 模式展示：
- [x] **进度时间轴**: 清晰展示 提交 -> 审核 -> 公示 -> 拨付 的关键节点及完成时间。
- [x] **政策说明**: 简要说明该补助的政策依据和适用范围。
- [x] **操作**: 提供查看政策原文的入口。
