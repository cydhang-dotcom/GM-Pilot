# 功能需求: 五险一金详情 (Social Security & Fund)

> **入口 ID**: `hr-4`
> **优先级**: P2
> **涉及页面**: `pages/workstation/hr/SocialSecurity.tsx`

## 1. 看板视图 (Level 2)
- [x] **总额卡片**: 
  - 醒目展示本月预计缴纳总额。
  - 拆分显示“公司承担”与“个人承担”金额。
- [x] **变动入口**: 
  - 统计本月新增/减少人数。
  - 点击进入变动明细 Overlay。
- [x] **凭证下载**: 列表展示社保/公积金的缴纳回单，支持下载。

## 2. 变动明细 (Level 3 Overlay)
- [x] **变动统计**: 顶部展示新增人数与减少人数卡片。
- [x] **人员清单**: 
  - 列表展示变动人员姓名、类型（新增/减少）。
  - 展示具体的缴费基数及公司/个人承担金额。
