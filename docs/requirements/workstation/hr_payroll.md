# 功能需求: 薪酬管理详情 (Payroll)

> **入口 ID**: `hr-1`
> **优先级**: P1
> **涉及页面**: `pages/workstation/hr/Payroll.tsx`

## 1. 看板视图 (Level 2)
- [x] **本月预览**: 
  - 展示饼图 (Pie Chart) 分析本月薪资构成（基本工资、绩效、社保、个税）。
  - 显示预计发放总额。
- [x] **历史列表**:
  - 按月展示历史工资表状态（待确认/已发放）。
  - **紧急状态**: 若有“待确认”且临近发薪日的记录，需通过左侧色条或动效进行视觉强调。
  - 操作: 提供“导出报表”快捷按钮。

## 2. 薪资详情 (Level 3 Overlay)
点击单月记录弹出全屏详情页。
- [x] **头部摘要**: 
  - 深色背景卡片展示实发总额。
  - 标记“五险一金已扣除”、“个税已申报”状态。
- [x] **人员明细**:
  - 列表展示每位员工的 实发金额 与 代扣税额。
  - 状态标签: 标记“试用期”员工。
- [x] **底部操作**:
  - **待确认状态**: 显示“反馈异议”与“确认并发放”双按钮。
  - **已发放状态**: 显示“下载工资条”按钮。
