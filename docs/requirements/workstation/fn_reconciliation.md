# 功能需求: 对账中心 (Reconciliation)

> **入口 ID**: `fn-rec`
> **优先级**: P0
> **定位**: 财务模块的核心入口，连接流水、发票、报销与凭证。
> **涉及页面**: `pages/workstation/finance/Reconciliation.tsx`

## 1. 核心看板 (Health Dashboard)
- [x] **健康度评分**: 展示同步健康度百分比（如 92%）。
- [x] **数据概览**: 展示本月流水笔数与待处理差异数。

## 2. 差异处理清单 (Action List)
- [x] **聚合展示**: 将同类异常（如“流水未解释”、“发票待归属”）聚合展示，并在右上角标记数量。
- [x] **优先级**: 紧急事项（Urgent）需用红色高亮图标。

## 3. 差异详情与沟通 (Level 3 Overlay)
- [x] **异常卡片**: 清晰描述异常原因（如“系统未找到对应合同”）。
- [x] **上下文数据**: 自动关联展示相关的流水或单据快照，辅助决策。
- [x] **沟通记录 (Chat)**:
  - 展示代理会计的留言/询问。
  - 提供底部输入框，允许总经理直接回复说明情况。
- [x] **一键处理**: 简单的异常提供“标记已处理”按钮。
