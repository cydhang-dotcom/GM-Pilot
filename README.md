# GM Pilot - 总经理驾驶舱

> **GM Pilot** 是专为中小企业总经理（GM）量身定制的移动端履约管理 App。通过聚合人事与财务外包服务数据，将复杂的账目与流程转化为直观的经营指标，助力管理者实现“上帝视角”的决策把控。

## 🚀 核心价值 (Core Values)

- **经营透明**: 实时掌握盈亏状况、现金流走势及支出结构。
- **履约监控**: 数字化追踪外包服务商（HR/财务）的交付进度，确保节点不延误。
- **隐私至上**: 针对移动端办公场景，核心财务数据（如余额、薪资）默认应用高斯模糊保护。
- **极简决策**: 所有的复杂逻辑隐藏在后台，GM 只需进行关键节点的“确认”与“审批”。

## 🛠 功能模块 (Functional Modules)

### 1. 经营总览 (Dashboard)
- **盈亏看板**: 自动计算净利润与利润率，支持红绿动态主题。
- **AI 智能诊断**: 基于大模型对财务数据进行深度扫描，输出合规风险与经营建议。
- **资金心脏**: 对比本月与上月现金流消耗速度，预警流动性风险。

### 2. 智能待办 (Inbox)
- **交付时间轴**: 像素级对齐的 Timeline，展示发薪、报税、关账等关键节点。
- **任务聚合**: 智能聚合琐碎的 OA 审批与入职任务，减少信息噪音。
- **置顶账单**: 待支付服务费账单强制置顶，确保服务不中断。

### 3. 事务工作台 (Workstation)
- **人事管理**: 薪酬明细、员工档案、五险一金变动及合同到期预警。
- **财务运营**: 银行流水解释、发票采集、报销合规预检及电子凭证管理。
- **综合服务**: 专属服务顾问一键直达，政策咨询与异常处理工单化追踪。

### 4. 企业中心 (Company)
- **数字名片**: 企业基本资料与信用代码一键复制。
- **资产管家**: 电子印章状态实时监控，数字证书有效期预警。
