# GM Pilot - 总经理驾驶舱

> **GM Pilot** 是专为中小企业总经理（GM）量身定制的移动端履约管理 App。通过聚合人事与财务外包服务数据，将复杂的账目与流程转化为直观的经营指标，助力管理者实现“上帝视角”的决策把控。

## 🚀 核心特性 (v2.2.0)

- **AI 智能诊断**: 集成 Gemini Pro API，对本月财务数据、历史盈亏进行多维度扫描，输出具有专业深度的经营建议与风险预警。
- **穿透式审计 (Drill-down)**: 从薪资总额一键点进个人工资条，从记账凭证直接穿透到原始回单与发票，确保账目透明无死角。
- **履约动态追踪**: 实时查看五险一金增减员办理进度，明确展示“预计办理日期”及“办理中”动效。
- **20px 轴心法则**: 严格遵循像素级对齐的设计系统，提供极致的秩序感与移动端阅读体验。
- **隐私心脏**: 敏感财务数值（余额、个税、基本工资）默认高斯模糊，支持原位快速切换显示。

## 🛠 功能模块 (Functional Modules)

### 1. 经营总览 (Dashboard)
- **盈亏看板**: 净利润实时计算，支持盈利（Emerald）与亏损（Rose）动态主题切换。
- **资金心脏**: 同比/环比对齐的现金流消耗走势图，预警流动性风险。

### 2. 智能待办 (Inbox)
- **交付时间轴**: 像素级对齐的 Timeline，展示发薪、报税、月结等履约节点。
- **入职办理中心**: 支持员工扫码自助入职，强制 GM 补全岗职与薪资，实现合规入职。

### 3. 事务工作台 (Workstation)
- **财务对账中心**: 差异事项清单化管理，支持与代理会计在线即时沟通。
- **员工全景档案**: 360度展示任职、合同、教育及工作历史，支持移动端直接修改。
